pub struct ShareableOrderedVec<T> { /* fields omitted */ }
Expand description

A collection that keeps the ordering of its elements, even when deleting an element However, this collection can be shared between threads We can guess what the index is for an element that we must add We can use get, and get_next_id_increment on other threads, but that is all We must do the rest of our operations using an external messaging system
